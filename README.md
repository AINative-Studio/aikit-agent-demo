# agent-browser-demo

Full-featured chat application with Claude AI

## Features

- App Router
- Server Components
- Streaming responses
- Chat history
- Tailwind CSS
- TypeScript

## Getting Started

### Prerequisites

- Node.js 18 or higher
- 

### Installation

```bash
pnpm install
```

### Environment Variables

Copy `.env.example` to `.env` and fill in your values:

```bash
cp .env.example .env
```

Required environment variables:
- `ANTHROPIC_API_KEY`

### Development

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Test

```bash
pnpm test
```
